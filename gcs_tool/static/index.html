To connect MAVLink:

  1. Start SITL simulation:
    sim_vehicle.py -v ArduCopter --console --map
  2. Start LLM Server:
    python server.py
  3. Start GCS Server (connects automatically to SITL):
    python gcs_server.py
  4. Telemetry will appear below when connected

Or start GCS server with custom MAVLink connection:

python gcs_server.py --mavlink=udpin:0.0.0.0:14550
